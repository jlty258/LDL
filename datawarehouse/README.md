# 制造业数据仓库项目

## 项目概述
这是一个完整的制造业数据仓库项目，包含ODS、DWD、DWS、ADS四层数据架构。

## 数据仓库分层

### ODS层（操作数据存储层）
- 30张源数据表，存储来自业务系统的原始数据
- 最大表：10000行数据

### DWD层（数据明细层）
- 对ODS层数据进行清洗、转换、标准化
- 保持与ODS层相同的粒度

### DWS层（数据汇总层）
- 基于DWD层数据进行轻度汇总
- 按业务主题进行数据聚合

### ADS层（应用数据服务层）
- 面向应用的数据集市
- 提供业务报表和分析所需的数据

## 调度任务
- DolphinScheduler: 30个调度任务
- Airflow: 30个调度任务
